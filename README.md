## Descripción

Este repositorio contiene el desarrollo de la **Parte II del Parcial 1** de la asignatura **Análisis y Diseño de Algoritmos I**.

Como parte de la evaluación se seleccionaron y resolvieron dos ejercicios de HackerRank pertenecientes a las categorías solicitadas por el docente:

- **Sorting:** Big Sorting
- **String:** Super Reduced String

Para cada ejercicio se realizó:

- Implementación en C++.
- Análisis de complejidad temporal y espacial.
- Pruebas de funcionamiento.
- Documentación mediante README.
- Sustentación en video.
